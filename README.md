Atividade Realizada para a disciplina de Segurança da Informação

Esteganografia deriva do grego, onde estegano significa esconder, mascarar e grafia significa escrita. Logo, esteganografia é a arte da escrita encoberta.

# Terminologia

- **Dado embutido (Embedded Data)**: é o dado que será enviado de maneira secreta normalmente em uma mensagem, texto ou figura. Representado no projeto através do arquivo "messageInputFile.txt"
- **Mensagem de Cobertura ou *Cover Message* : é a mensagem que servirá para mascarar o dado embutido. Esta mensagem de cobertura pode ser áudio (*cover-audio*), de texto (*cover-text*) ou uma imagem (*cover-image*). Representado pelo arquivo "inputImage.png"
- **Estego-objeto ou stego-object**: após a inserção do dado embutido na mensagem de cobertura se obtém o estego-objeto. Representado pelo arquivo "encoded_img.png".
